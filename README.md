# Walert - A Conversational Agent
Demo Video Link: https://bit.ly/chiir24walertdemovideo

We built Walert, a conversational agent that answers FAQs about programs of study that are offered in the School of Computing Technologies at RMIT University. This intent-based approach, deployed in Amazon Echo device, was showcased as a demo at RMIT Universityâ€™s Open Day in August 2023.
